## 💬 Usage
`$ git clone https://github.com/Keita0428m/ToDo-Card-App.git` <br>
`$ cd ToDo-Card-App` <br>
`$ bundle install` <br>
`$ rails db:create` <br>
`$ rails db:migrate` <br>
`$ rails s `<br>
`👉 http://localhost:3000 `
